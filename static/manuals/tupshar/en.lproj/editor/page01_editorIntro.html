Overview

Tupshar's editor window is where you will do most of your work. It is split into two panes: the text view pane on the left, and the editor pane on the right. The text view pane lets you see a preview of your text and make selections for editing or deletion; the edit pane lets you add lemmas, and make edits.

1 Text representation switcher Switches between a cuneiform, transliteration, normalisation, and word-by-word translation view of the underlying document.
2 Text view Displays a view of the text. This is not editable directly, but in the transliteration and normalisation views individual words may be selected, and edited in the Edit pane.
3 Edit pane Main interface for inputting and editing words in the document, or adding document properties such as a title and author.
4 Lemma editor Input new lemmas, or modify existing ones, using these three boxes.
5 Add lemma Performs the editing operation described in the the status bar (6), the default being to add to the end of the document.
6 Status bar Describes the current position of the insertion point, as well as whether the next input operation will append a new lemma to the end of the document, overwrite the selected lemma, or insert a new lemma in the middle of the document.
7 Document translation box Add an accompanying idiomatic translation to the document here.
