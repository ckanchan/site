Navigating Catalogues

The first window you will see when SAAo-SX loads is the Catalogue View. This is a simple table listing all the text editions available in a given catalogue file, usually corresponding to either to a single published volume of related texts or to an Oracc project.

1 Text List Lists all texts in the catalogue, showing designation, friendly title, and author if available
2 Volume selection dropdown By default this uses the builtin database, but can download volumes from Oracc.
3 Bookmarks button Toggle between showing all texts or bookmarked texts in the table. Bookmarked texts are indicated by their bold entries
4 Info Describes how the current catalogue was retrieved, either downloaded from an online source, loaded from the builtin local database, from a local file, or from the bookmark list. Defaults to the builtin local database.
5 Glossary button Opens the Glossary window
6 View online Opens the Oracc.org web page for the currently selected text in your default browser.
7 Search box Searches texts in the currently loaded catalogue by designation, title or P/X number.
8 Info sidebar Shows information about the currently selected text, including its text IDs in various cataloguing systems, publication history, and archaeological data. This sidebar can be hidden with the shortcut ⌃⌘S for a compact view.
