Reading Texts

The text viewer is the main interface for reading cuneiform texts. It provides additional navigational affordances for moving through texts as well.

1 Text Pane Shows a representation of the cuneiform text. The tab control at the top switches between cuneified, transliterated, normalised views and translations into English. A single-paned window can optionally be set to open in the Preferences.
2 New Window button Opens a new window showing the same text. If viewing a search set (e.g. from a glossary lookup) then the new window will also be linked to that search set.
3 Glossary button Opens the builtin glossary.
4 Catalogue Search dropdown Searches the text's catalogue and displays results in a popover.
5 Bookmark button Adds or removes the text from the user's bookmarks.
6 Get Info Opens a panel with catalogue entry data on the text. This is the same as the info shown in the Catalogue View's right pane.
7 View online Opens the Oracc.org web page for the currently selected text in your default browser.
8 Map View button Plots the places mentioned in the text on a map. Beta functionality.
9 Quick define bar Shows a glossary definition for the word selected in the text pane. Currently only available in the Transliteration view.
