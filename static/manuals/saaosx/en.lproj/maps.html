The Map View

The map viewer plots the locations of ancient places on a map, and displays details for them if available. When accessed from the catalogue window, it shows the locations of all ancient places within the SAAo index of place names. When accessed from the text view screen, it filters the texts for place names and then plots those on a map, together with the excavation site for the letter, if possible. The map view is still beta functionality

1 Map The main map view, here shown in dark mode. Clicking on a pin shows the name of the location, as well as selecting it in the table and showing a description if available.
2 Description If available, shows a description of the site from the Pleiades online linked database of ancient places.
3 Location list A list of all locations in the SAAo glossary. Currently this appears in a nondetermined order upon every load
