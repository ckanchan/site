Keyboard shortcuts

Catalogue View

⇧⌘N, ⌘N Opens a new catalogue window with the currently loaded catalogue. If there is no window on screen, it opens to the builtin database.
⇧⌘G Opens a new glossary window.
⌘O Opens the file selection dialog.
⌃⌘S Toggles the sidebar.

Text View

⌘N Opens a new text window.
⌘L Sets focus to the Catalogue Search field for quick searching entries
⌘→ Moves to the next text in the text set from which the current text has been loaded, either the next text in the catalogue or the next text in the search results.
⌘← Moves to the previous text in the text set from which the current text has been loaded, either the previous text in the catalogue or the previous text in the search results.
