First Steps

Tupšenna is an iOS app designed to provide a fast, offline-first interface to the State Archives of Assyria projects and other cuneiform texts. It has three main interfaces:

Catalogue

The catalogue view is the first view you will see when opening Tupšenna for the first time. It shows all the texts you have downloaded, and you can tap on any name to open that text in the text view. You can download individual 'volumes' of the SAA project and selected other Oracc projects from the Preferences screen. Read more

Text Viewer

The text viewer displays cuneiform texts. It is designed to make it easy to view a text in any of its possible representations: as a transliteration, normalisation, translation, or 'cuneified' approximation. It loads texts selected from the Catalogue window. Read more

Glossary

Browse the computer-generated SAAo glossary, and look up all texts which contain a given lemma. Read more
