Reading Texts

The text viewer is the main interface for reading cuneiform texts. It provides additional navigational affordances for moving through texts as well.

1 Info Opens the Text Info view which shows metadata about the text, including publication numbers, catalogue numbers, and a link to view the text on the Web.
2 Text View Displays the text. You can change the representation of the text you're viewing by using the switch (6) at the bottom of the view.
3 Highlight for Quick Define Using the standard iOS text selection gestures, you can select a word in the transliteration and normalisation views and see a quick definition in the toolbar at the bottom of the screen (4).
4 Quick Define Shows the definition for the most recently highlighted word
5 Quick Navigate Lets you quickly move to the previous or next text in project catalogue order.
6 Select View Switch between cuneified, transliterated, normalised and translated views of the document here
