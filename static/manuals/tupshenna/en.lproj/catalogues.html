Navigating Catalogues

The first window you will see when Tup≈°enna loads is the Catalogue View. This is a simple table listing all the text editions available in a given catalogue file, usually corresponding to either to a single published volume of related texts or to an Oracc project.

1 Preferences Opens the Preferences screen, where you can download volumes for reading.
2 Help Loads this online help
3 Text List Shows all the texts available for reading. Tap on a text item to read it
4 Glossary Opens the built-in glossary.
