Using the Glossary

The built in glossary provides support for looking up translations for the Akkadian terms used in the accompanying texts, as well as searching for all instances of a given term. Unlike the Oracc interface, it does not yet offer the ability to perform more refined lookups, such as by written or normalised forms.

1 Search Search the glossary for entries. You can use the scope selector (2) to control whether to search Akkadian only, English translation only, or all fields.
2 Scope selection Limit the scope of the search to exact Akkadian match, English match, or fuzzy search in al fields
3 Lemma list When a search is active, shows search results. Otherwise, displays the entire glossary.
4 Transliteration characters Allows you to easily enter special characters used in Akkadian transliteration.
